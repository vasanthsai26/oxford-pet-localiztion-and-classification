About The Model

  • The LocNet Model has ResNet 50 backbone with a custom bounding box head and classification head. This object localization model has trained on Oxford Pets dataset.
  • The Oxford Pets dataset (also known as the "dogs vs cats" dataset) is a collection of images and annotations labeling various breeds of dogs and cats.
  • There are approximately 100 examples of each of the 37 breeds. This dataset contains the object detection portion of the original dataset with bounding boxes around the animals' heads.
